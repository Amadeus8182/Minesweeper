# Minesweeper
Minesweeper written in Java using Swing.  
<img src="./demo.png" width="500" height="341">

## Play
To play using the graphical interface:
```
javac MinesweeperGUI.java
java MinesweeperGUI
```

## Terminal
To play using the terminal:
```
javac Minesweeper.java
java Minesweeper
```
